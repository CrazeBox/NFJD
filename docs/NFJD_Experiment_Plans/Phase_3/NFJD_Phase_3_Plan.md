# NFJD Phase 3: 高冲突数据验证计划书

## 1. 阶段目标

在高冲突合成数据上验证NFJD相比FedJD和基线方法的优势，证明NFJD在目标间存在真实冲突时仍能有效优化。

## 2. 数据集

| 数据集 | 冲突强度 | m | 余弦相似度 |
|--------|----------|---|-----------|
| HighConflict cs=0.5 | 中等 | 2,3,5 | ≈-0.5 |
| HighConflict cs=1.0 | 强 | 2,3,5 | ≈-0.96 |
| HighConflict cs=2.0 | 极强 | 2,3,5 | ≈-0.99 |

## 3. 实验矩阵

5方法 × 3个m值 × 3冲突强度 × 3种子 = **135次实验**

## 4. 关键验证点

1. NFJD在高冲突下Avg RI仍 > FedJD
2. 冲突强度增加时，NFJD的RI下降幅度 < FedJD
3. NFJD的NHV在高冲突下仍 > 0.9
4. NFJD的通信优势在高冲突下不变

## 5. 通过标准

1. NFJD在所有冲突强度下Avg RI > FedJD
2. NFJD在cs=2.0时RI下降 < 10%（vs FedJD下降 > 20%）
3. 所有实验无NaN/Inf
